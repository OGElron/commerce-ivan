## Deployment

To deploy this project run

```bash
  npm run deploy
```


## Features

- Product catalog in cards format for an e-commerce
- Dynamic categories 
- Catalog and categories are done from database (no mock)
- Cart standard functions: add, remove, empty, total.
- Navigable menu
- Made with React-Bootstrap
